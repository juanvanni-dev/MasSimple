# Más Simple SF 🥗

A high-conversion landing page with a full shopping cart, WhatsApp order integration and a custom admin dashboard — built for a real healthy frozen food business in Santa Fe, Argentina.

**[→ Live Site](https://mas-simple.vercel.app/)**

---

---

## What It Does

- Browse and select healthy frozen food products
- Add items to a shopping cart with quantity control
- Place orders directly via WhatsApp with a pre-filled message
- Admin dashboard to manage products, stock levels and incoming orders — no backend required
